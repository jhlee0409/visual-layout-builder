# Legacy Code Archive

이 디렉토리는 Laylder V1의 이전 코드를 보관합니다.

## 구조

- `app/` - V1 메인 페이지 및 V2 개발 페이지
- `components/` - V1 컴포넌트들
  - `breakpoint-panel/` - V1 브레이크포인트 패널
  - `component-panel/` - V1 컴포넌트 패널
  - `generation-modal/` - V1 생성 모달
  - `grid-canvas/` - V1 그리드 캔버스
- `lib/` - V1 라이브러리 파일들
- `store/` - V1 상태 관리
- `types/` - V1 타입 정의

## 마이그레이션 이력

현재 메인 버전은 V2(Component Independence 아키텍처)입니다.
V1 코드는 참조용으로만 유지됩니다.
